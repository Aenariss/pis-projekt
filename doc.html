    Postup instalace

    1. Instalace závislosti: java 17 (např. openjdk-17-jdk), maven
    2. Instalace závislostí frontendu pomocí příkazu: mvn process-resources
    3. Instalace dalších závislostí a spuštění serveru pomocí příkazu: mvn liberty:run
    4. Inicializace databáze: otevření url http://localhost:9089/api/prefill v prohlížeči (GET)
    5. Nyní je možné webovou aplikaci používat na url http://localhost:9089/
